Virtual Desktop PlayMode
==========================================================================================

This script allows a user to develop entirely in VR with Unity and Virtual Desktop 
by configuring the Play Mode window to be launched on the Rift.

The script can be used without Virtual Desktop as well to quickly preview changes in VR.

Requirements:
- Oculus Unity Integration package imported in your project
- An OVRPlayerController or OVRCameraRig prefab placed in your scene

Installation:
- Import the package into a project that meets the requirements above
- Press Ctrl-P
